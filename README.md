# kolpa-docs Minimal Starter

Starter template for [kolpa-docs](https://github.com/kolpa-engine/kolpa-docs).

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install

# bun
bun install
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev
```


